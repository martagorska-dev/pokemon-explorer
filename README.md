# Pokemon Explorer

## Description

**Pokemon Explorer** is a web app for browsing Pokémon by type using [PokeApi](https://pokeapi.co/). Features:

- Pokémon cards with thumbnail, name, number, types, and colored backgrounds.
- Filter by type (multi-type selection).
- Search Pokémon by name.
- Modal with details: sprites, evolutions, locations.
- Dynamic loading ("Load more").
- Favorites saved in localStorage, responsive design, semantic HTML + ARIA.

## Technologies

React | TailwindCSS / MUI | fetch + [TanStack Query](https://tanstack.com/query/latest) | React Router | Vite

## Installation

```bash
git clone https://github.com/martagorska-dev/pokemon-explorer
cd pokemon-explorer
npm install
npm run dev         # start dev server
npm run build       # build production
npm run preview     # preview build
```
